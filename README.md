# Kizuna 絆

> A bond between AI agents.

A plugin-based local long-term memory MCP server for Claude Code,
designed for cross-repository agent collaboration.

> ⚠️ **Status: Personal project, work in progress**
>
> This project is in early development. Public release is planned
> after Phase 4 completion. The repository may be force-pushed,
> rewritten, or restructured at any time during development.
> Issues and PRs may not receive responses. Use at your own risk.
